## Рабочая директория

Shell стартует в корне проекта — директории с этим файлом.
- `git` команды: запускать из корня проекта.
- Компонентные команды: `cd backend && ...` (использовать относительные пути).
- **Никогда** не использовать абсолютные пути.

---

## Проект

**Краткое описание:** Веб-приложение для расчёта стоимости трансфера до горнолыжных курортов
**Стек:** React + Vite + TypeScript (фронтенд), FastAPI Python (бэкенд)

| Компонент | Путь | Стек |
|-----------|------|------|
| Backend | `backend/` | FastAPI + Python |
| Frontend | `frontend/` | React + Vite + TypeScript |
