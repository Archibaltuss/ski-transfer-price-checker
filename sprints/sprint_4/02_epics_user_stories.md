# Эпики и User Stories — Sprint 4

---

## EPIC 4.1: Страница результатов

Цель: показать туристу результат расчёта с ценой и призывом к действию.

---

### US 4.1.1 — Отображение результата расчёта

**Как** турист,
**я хочу** видеть стоимость трансфера, название маршрута и количество пассажиров после расчёта,
**чтобы** принять решение о заказе трансфера.

**Агент:** `[frontend]`

Зависит от: US 3.1.3 (onResult prop передаёт данные)

Acceptance criteria:
- [ ] Компонент `ResultsPage` отображается вместо `SearchForm` после успешного расчёта
- [ ] Показывает: `price_chf` (форматирован как «CHF 387.50»), `route_name`, `distance_km`, `passengers`
- [ ] Кнопка «← Back» сбрасывает результат и возвращает к `SearchForm`

---

### US 4.1.2 — CTA "Get a real quote"

**Как** турист, заинтересованный в трансфере,
**я хочу** иметь возможность запросить реальное предложение,
**чтобы** перейти к фактическому бронированию.

**Агент:** `[frontend]`

Зависит от: US 4.1.1

Acceptance criteria:
- [ ] Кнопка «Get a real quote» присутствует на `ResultsPage`
- [ ] Кнопка открывает внешнюю ссылку или `mailto:` в новой вкладке
- [ ] Визуально выделена как основной CTA (контрастный цвет)

---

## EPIC 4.2: Деплой на продакшн

Цель: опубликовать рабочее приложение по публичному URL.

---

### US 4.2.1 — Деплой фронтенда на Vercel

**Как** разработчик,
**я хочу** чтобы фронтенд был доступен по публичному URL Vercel,
**чтобы** поделиться ссылкой в портфолио.

**Агент:** `[frontend]`

Зависит от: US 4.2.2 (URL бэкенда нужен для конфигурации)

Acceptance criteria:
- [ ] Репозиторий подключён к Vercel, деплой срабатывает автоматически на push в main
- [ ] `vercel.json` содержит rewrite: `/api/*` → `{BACKEND_URL}/api/*`
- [ ] `VITE_API_URL` (если используется) задан через Vercel Environment Variables
- [ ] Публичный URL Vercel открывается, форма отображается

---

### US 4.2.2 — Деплой бэкенда на Railway/Render

**Как** разработчик,
**я хочу** чтобы FastAPI-бэкенд был доступен по публичному URL,
**чтобы** Vercel мог проксировать API-запросы к нему.

**Агент:** `[backend]`

Зависит от: нет

Acceptance criteria:
- [ ] Бэкенд задеплоен на Railway или Render из папки `backend/`
- [ ] Команда запуска: `uvicorn main:app --host 0.0.0.0 --port $PORT`
- [ ] `GET {BACKEND_URL}/api/health` возвращает HTTP 200 из интернета
- [ ] CORS настроен: разрешён домен Vercel (`*.vercel.app`)
- [ ] `backend/requirements.txt` актуален (все зависимости перечислены)
